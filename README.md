## Environment Configuration

Set deployment-specific values through your local `.env` file, hosting provider environment settings, or a secret manager. Do not commit real secret values.

- `ADMIN_PASSWORD`: login password for the admin session endpoint.
- `ADMIN_SESSION_SECRET`: HMAC signing key for admin session cookies. If omitted, the app falls back to `ADMIN_PASSWORD` for compatibility with existing deployments. Setting or rotating this value invalidates existing admin sessions.
- `SMS_WEBHOOK_SECRET`: shared secret required by the SMS webhook endpoint.
- `DATABASE_URL`: Prisma database connection string.
- `HERO_API_KEY`: HeroSMS API key. If this is stored in the database through the admin settings flow, keep database backups and admin access protected; otherwise store it in environment configuration or a secret manager.
